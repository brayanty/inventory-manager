# Invertory manager

## How to install

Clone the project

```bash
  git clone https://github.com/brayanty/inventory-manager.git
```

Go to the project directory

```bash
  cd inventory-manager
```

Start the server

```bash
  docker compose run --build
```

## Frontend main techs

Use Vite, React, Tailwindcss, DaisyUI

## Backend main techs

NodeJS, ExpressJS

## Printer main techs

NodeJS, ExpressJS
